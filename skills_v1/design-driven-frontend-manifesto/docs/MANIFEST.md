# DDFM Sentient Manifest v5.0

## 🎨 Surgical Aesthetics
### 1. Color Sovereignty (OKLCH)
- **Standard**: Never use HEX/RGB. Use OKLCH for perceptual linearity.
- **Contrast**: Automatically adjust L-value based on background luminance to ensure WCAG AAA.

### 2. Phi-Based Spacing
- **Geometry**: Spacing tokens (4, 8, 12, 20, 32, 52) must follow the Golden Ratio approximation for visual harmony.

## ⚡ Kinetic Velocity (VAK)
### 1. Emotional Kinetics
- **frustration_aware**: UI detect multi-clicks or "rage-clicks" and automatically slows down animations + adds helpful tooltips.
- **Physics**: Use Spring physics (Stiffness: 400, Damping: 30) for "Magnetic" effects.

### 2. Velocity-Aware Interaction
- **Ghost Motion**: Elements should trail slightly when dragged, indicating predictive velocity.

## 🛠️ Precision Engineering
### 1. State-Chain Mastery
- **XState**: Complex UI flows (Multi-step forms, Auth) must use State Machines. Zero-boolean logic.
- **Optimistic UI**: Every write action assumes success; rollback with "Ghost transitions."

### 2. Hardware-Aware Rendering
- **Performance**: Detect low-power mode or high CPU. Automatically degrade blur/shadows/3D to maintain 60FPS.
- **Zero-CLS**: Strict aspect-ratio containers for all media/dynamic content.

## 🛡️ Visual Self-Evolution
- **Visual Audit**: FDFM uses `browser_subagent` to visually verify the design against the "Aesthetic Scorecard" (Balance, Contrast, Density).
